## 1. 現状分析

### 1.1 現在のダッシュボード構成

| セクション | 内容 | 問題点 |
|-----------|------|--------|
| 3D Car Attitude | 車両の3Dモデル + 姿勢角 | 実用性低い、視認性悪い |
| Speed/RPM/Gear | 数値表示 | OK |
| Throttle/Brake | バーグラフ | OK |
| Tyre Info | 4輪の温度・サスペンション | 見にくい |
| Lap Info | ラップタイム | OK |
| Position | XYZ座標 | ドライバーには不要 |
| Charts | 速度・RPMチャート | 悪くない |

### 1.2 実際のレーシングダッシュボードとの比較

| 項目 | 現在のGT7 Tool | 本物のレース | 差分 |
|------|---------------|-------------|------|
| ギア表示 | 小さい数値 | 大きな中央表示 | ❌ |
| シフトライト | なし | 必須 | ❌ |
| Gメーター | なし | 重要 | ❌ |
| タイヤ温度 | 数値のみ | 色分けグラフ | △ |
| 燃料残量 | 数値のみ | 周回数表示 | △ |
| ブレーキバイアス | なし | 重要 | ❌ |
| ドライバーペダル | なし | あると良い | ❌ |

## 2. 提案：レーシングスタイルダッシュボード

### 2.1 コンセプト

**「F1 / WEC スタイルのテレメトリ表示」**

- ドライバーが走行中に見る情報に特化
- ピットウォール/エンジニアが見る分析情報は別タブ
- 高コントラスト・大文字・直感的な表示

### 2.3 各コンポーネント詳細

#### A. ギア & シフトライト（中央最上部）
```
┌─────────────────┐
│  ● ● ● ○ ○ ○  │  ← シフトライト（RPM連動）
│       3        │  ← ギア（巨大表示）
│    7,850 RPM   │  ← RPM（小さめ）
└─────────────────┘
```
- ギア: 200px以上の巨大フォント
- シフトライト: 6-8個のLED風ドット
- 色: 緑→黄→赤（RPMに応じて点灯）

#### B. 速度（右上）
```
┌───────────┐
│   287     │
│  km/h     │
└───────────┘
```
- 大きなデジタル表示
- 0-400km/h対応

#### C. スロットル & ブレーキ バー
```
┌─────────────────────────────────────────┐
│  THROTTLE                              │
│  ████████████████████████████ 100%     │
│                                         │
│  BRAKE                                 │
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░ 0%       │
└─────────────────────────────────────────┘
```
- 横長のバーグラフ
- スロットル: 緑/青
- ブレーキ: 赤

#### D. G-Force メーター
```
┌───────────────┐
│  G-FORCE      │
│      ▲        │
│   ◄  ●  ►    │
│      ▼        │
│  L: 1.2 R:0.3│
└───────────────┘
```
- 円形のGメーター
- 横G（コーナリング）と縦G（加速/減速）
- ドットがリアルタイムで移動

#### E. タイヤ温度（4輪）
```
┌───────────────┐
│  TYRE TEMP    │
│  FL   FR      │
│  87°C 89°C   │
│  RL   RR      │
│  85°C 88°C   │
└───────────────┘
```
- 色分け: 青（冷たい）→ 緑（最適）→ 赤（過熱）
- 4輪の配置を視覚化

#### F. ラップ情報
```
┌───────────────────┐
│  LAP 4/10         │
│  CURRENT 1:23.456 │
│  BEST    1:22.123 │
│  DELTA   -0.234   │
└───────────────────┘
```
- 現在ラップ / 総ラップ
- 現在タイム
- ベストタイム
- デルタ（ベストとの差）

#### G. 燃料
```
┌─────────────────────────────────────────┐
│  FUEL  ████████████████░░░░  65L       │
│         5.2 laps remaining             │
└─────────────────────────────────────────┘
```
- 燃料残量（バー + 数値）
- 残り周回数

## 5. 代替案

### 案A: SimHubスタイル
- よりカスタマイズ可能
- 複数ページ切り替え
- 工数: +10h

### 案B: MoTec i2スタイル
- 分析重視
- ポストセッション分析向け
- 工数: +20h

### 案C: F1ゲームスタイル（推奨）
- シンプルで視認性良好
- 実装コスト低
- 工数: 8-12h
