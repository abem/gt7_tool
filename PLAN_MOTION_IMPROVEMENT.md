# 3D車両モデル 挙動改善計画書
## 「ブルブルしているだけでわからない」問題の解決（第3版）

---

## 1. 現状の問題点（第3版）

### 1.1 ユーザー報告（最新）
> - バウンスを有効にすると「ちらつくだけ」になる
> - 増幅が大きすぎると「裏返るほど回転する」
> - タイヤの切れ角が90度を超えることがある

### 1.2 問題分析

| 問題 | 原因 | 対策 |
|------|------|------|
| **バウンスでちらつく** | stiffness/dampingの比率が不適切 | バウンス無効化、慣性のみ使用 |
| **裏返るほど回転** | 増幅値が大きすぎる（20/15） | 増幅を控えめに（3.0/2.5） |
| **タイヤが90度以上** | steering増幅が1.0（実寸） | 増幅を0.3に下げる |

---

## 2. 採用した解決策

### 2.1 コンセプト
**「振動なしで、滑らかに、微妙な傾きが分かる動き」**

- バウンス（バネ）は**無効**（ちらつき防止）
- 慣性（lerp）のみで**滑らかに追従**
- 増幅は**控えめ**（裏返らない程度）
- ステアリングは**1/3に抑制**

### 2.2 期待する動き

| シチュエーション | 期待動作 |
|-----------------|----------|
| **急ブレーキ** | 車体が滑らかに前傾（振動なし） |
| **急加速** | 車体が滑らかに後傾 |
| **コーナリング** | 車体が外側に傾く（ロール） |
| **直進** | ほぼ静止（振動なし） |

## 3. 現在のパラメータ（第3版）

### 3.1 実装済みパラメータ

```javascript
var EXAGGERATION_CONFIG = {
    enabled: true,

    // 増幅設定（控えめに）
    amplification: {
        pitch: 3.0,    // ブレーキ/加速の傾き（控えめ）
        roll: 2.5      // コーナリングの傾き（控えめ）
    },

    // バウンス設定（無効化）
    bounce: {
        enabled: false,
        stiffness: 0,
        damping: 0
    },

    // 慣性設定（滑らかな追従）
    inertia: {
        enabled: true,
        lerpFactor: 0.3  // 滑らかに追従
    },

    // 微振動設定（無効）
    vibration: {
        enabled: false,
        baseAmplitude: 0,
        rpmMultiplier: 0,
        frequency: 0
    },

    // ステアリング設定（抑制）
    steering: {
        amplification: 0.3,     // 実寸の1/3
        lerpFactor: 0.2         // 滑らかに追従
    }
};
```

### 3.2 パラメータの狙い

| パラメータ | 値 | 狙い |
|-----------|---|------|
| pitch: 3.0 | 控えめ | ブレーキ/加速時の傾きを微妙に |
| roll: 2.5 | 控えめ | コーナリング時の傾きを微妙に |
| bounce.enabled | false | ちらつき防止 |
| inertia.enabled | true | 滑らかな追従 |
| inertia.lerpFactor: 0.3 | 中程度 | 速すぎず遅すぎず |
| steering.amplification: 0.3 | 抑制 | タイヤ切れ角を現実的に |

## 6. 今後の調整案

### 6.1 さらに調整が必要な場合

**動きが小さすぎる場合:**
amplification: { pitch: 5.0, roll: 4.0 }

**動きが大きすぎる場合:**
amplification: { pitch: 2.0, roll: 1.5 }

**追従が遅い場合:**
inertia: { lerpFactor: 0.5 }

**追従が速すぎる場合:**
inertia: { lerpFactor: 0.15 }
